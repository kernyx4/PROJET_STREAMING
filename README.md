# 🎬 NEF-DIVAS - Interface de Streaming

Bienvenue sur **NEF-DIVAS**, un projet de plateforme de streaming conçu et développé par l'équipe **Web Divas**. Ce site propose une interface moderne, inspirée des meilleurs services de vidéo à la demande.

## 👥 L'Équipe & Rôles
Nous sommes un groupe de trois étudiantes passionnées :
* **Hawa Kane** (Capitaine) : Responsable de la structure **HTML de l'accueil**.
* **Aïssatou Barry** : Responsable du design **CSS de l'accueil** et de l'identité visuelle.
* **Mame Diarra Bousso Kane** : Responsable du développement **HTML et CSS** des sections dédiées aux **Films et Séries**.

## 🚀 Fonctionnalités
- **Catalogue Dynamique** : Présentation de films et séries avec des miniatures organisées par catégories.
- **Expérience Immersive** : Intégration de photos et de vidéos pour un rendu réaliste.
- **Interactivité** : Utilisation d'effets de survol (hover) pour révéler les boutons de lecture et d'information.

## 🛠️ Spécifications Techniques
Ce projet a été réalisé exclusivement avec les technologies suivantes :
* **HTML5** : Pour une structure solide et sémantique.
* **CSS3** : Pour une mise en page avancée utilisant :
    - **CSS Grid** pour la disposition des affiches.
    - **Aspect-ratio** pour l'uniformité des visuels.
    - **Overlays** pour les éléments interactifs.

## 📖 Utilisation
Pour lancer le projet :
1. Téléchargez ou clonez le dépôt.
2. Ouvrez le fichier `index.html` dans votre navigateur.
